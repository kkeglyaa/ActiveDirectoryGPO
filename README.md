# ActiveDirectoryGPO
настройка объекта групповой политики
В данной лабараторной работе была выполненна настройка групповых политик в доменной среде Active Directory для управления пользовательскими ограничениями.
1. Создание групповых политик.
   В оснастке Group Policy Management были созданы новые объекты:
   - запрет панели управления.
   - запрет смены обоев рабочего стола.
   - запрет доступа командной строки.
2. Настройка политики.
   Запрет панели управления: Конфигурация пользователя -> Политики -> Административные шаблоны -> Панель управления.
   Запрет смены обоев рабочего стола:  Конфигурация пользователя -> Политики -> Административные шаблоны -> Персонализация.
   Запрет командной строки: Конфигурация пользователя -> Политики -> Административные шаблоны -> Система.

4. Привязка политик к подразделению.
   Созданные политики были привязаны к подразделениям:
   HR - запрет панели управления, запрет смены обоев рабочего стола, запрет доступа командной строки.
   IT - запрет смены обоев.
   Management - запрет смены обоев.

5. Вывод:
   В ходе лабараторной работы были созданы и настроены групповые политики Active Directory, которые позволяют централизованно ограничивать действия пользователей.
   Была реализована модель управления доступом, при которой политики применяются к конкретным подразделениям.
